【提出前チェックリスト（正社員化コース）】

## 1. 対象判定（制度要件）
- [ ] 転換日がキャリアアップ計画期間内にある
- [ ] 正社員の定義（賞与/退職金 + 昇給）が就業規則等で明確
- [ ] 非正規の雇用区分と賃金の差（額または計算方法の差）が就業規則等で確認できる
- [ ] 転換制度の「手続き・要件・実施時期」が客観的に規定されている
- [ ] 転換後に試用期間を設けていない（設けた場合は取扱いが変わる）
- [ ] 転換後、社会保険の適用要件を満たし、必要に応じて加入させている

## 2. 賃金（3%増額）
- [ ] 転換前後の比較方法（月給同士/支給形態変更/所定労働時間変更）を適切に選んだ
- [ ] 3.00%以上（四捨五入不可）を満たす
- [ ] 算定から除外すべき手当（実費弁償、変動、賞与等）を除外した
- [ ] 固定残業代の減額・時間相当数の減少がある場合の注意点を確認した

## 3. 申請期限
- [ ] 「6か月分の賃金を支給した日（実際の支払日）」を特定した
- [ ] その翌日から2か月以内に提出できる（郵送は到着日基準に注意）

## 4. 添付書類の品質
- [ ] 添付書類は原本または原本の複写（加工・転記・別紙作成は避ける）
- [ ] 日付・氏名・雇用区分・賃金締日/支払日の整合がとれている
- [ ] 就業規則（周知・届出）の要件を満たす
